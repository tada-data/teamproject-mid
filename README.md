# teamproject-mid

주제 : 코로나로 인한 OTT서비스를 이용한 영화 시청비율 증가와 영화 산업의 투자 계획 수립

코로나 전후로 영화관 이용비율의 증감과 OTT서비스를 이용한 영화 시청비율의 증감을 확인하여 영화 산업에서 OTT서비스를 어떻게 이용해야 할 지 확인한다.

데이터 : KOSIS-문체부, 미디어 통계포털, KOBIS 참고
